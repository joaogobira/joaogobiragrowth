Como exportar como imagem
01
Abra o arquivo HTML no Google Chrome (funciona melhor que outros navegadores)
02
Pressione F12 para abrir o DevTools
03
Pressione Ctrl+Shift+P (ou Cmd+Shift+P no Mac) para abrir o painel de comandos
04
Digite "capture" e selecione Capture screenshot — captura exatamente o que está visível
05
Alternativa para banners grandes: No DevTools, clique no ícone de dispositivo (Ctrl+Shift+M), defina a dimensão manualmente e capture a tela cheia com Capture full size screenshot
06
Para editar o texto de qualquer banner: abra o arquivo .html no VS Code ou Bloco de Notas, localize as classes .big-headline, .tagline ou .subhead e altere o texto diretamente
